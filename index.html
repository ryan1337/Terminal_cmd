Commande du Terminal linux

Commande du bash

uname - a Affiche le systeme et kernel
date Affiche la date du system
whoami Affiche le nom d'utilisateru
man Affiche le manuel

Raccourcis bash

Ctrl + C Stop la commande courante
Ctrl + A Va en debut de ligne
Ctrl + E Va en fin de ligne
Ctrl + U Coupe du debut de la ligne
Ctrl + K Coupe de la fin de la ligne
Ctrl + R Recherche
!! Repete la derniere commande

ls Options

-a Affiche tous meme les fichiers caché
-R Liste recursive
-r Inverse L'ordre
-t Trie par dernier fichier modifié
-S Trie par taille de fichier
-l Longue liste
-1 Un fichier par ligne

Operations de dossier

pwd Affiche les dossier courant
mkdir Creé un nouveau dossier
cd Change de dossier
cd .. Remonte au dossier precedent
ls liste les fichiers

Permissions de fichier

chmod 775 fichier Change les permissions du fichier en 775
chmod -R 600 folder Dossier recursive 600

Numero de permission des fichiers

4 read (r)
2 write (w)
1 execute (e)
